Depth First Search: Breadth First Search:

In a depth-first crawl, the program will start at the start page, In a breadth-first crawl, the program will follow ALL links from the start page,

randomly choose one of the links on that page, then follow it to the next page. and ALL links from each page it visits, until the crawler has reached the limit of pages deep,

Then, at the next page, it randomly chooses a link from the options available, the options available (as measured from the start page) it should visit.

and follows it. This makes a chain from the starting page.



Keyword Search:

The keyword search allows you to search the webpages

for a specific keyword of your choice. When the word

is found, the program will halt and the page it was

found on will be highlighted in the graphs.
